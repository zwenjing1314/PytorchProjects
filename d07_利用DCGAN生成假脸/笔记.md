## 训练的过程
```text
您的理解：
1. 真数据 → 判别器 → 计算损失 → 反向传播 ✓
2. 假数据（detach）→ 判别器 → 计算损失 → 反向传播 → 更新判别器 ✓
3. 假数据 → 生成器 → 计算损失 → 反向传播 → 更新生成器 ✗（这里有点误解）

实际流程：
1. 真数据 → 判别器 → 计算损失 → 反向传播（累积梯度）✓
2. 假数据（detach）→ 判别器 → 计算损失 → 反向传播（累积梯度）→ 更新判别器 ✓
3. 假数据 → 判别器 → 计算损失 → 反向传播 → 更新生成器 ✓
   ↑ 
   关键：假数据通过判别器回传梯度到生成器，不是直接放入生成器
```
